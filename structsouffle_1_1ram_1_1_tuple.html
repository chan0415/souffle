Detailed Description

template<typename Domain, std::size_t _arity>
struct souffle::ram::Tuple< Domain, _arity >

The type of object stored within relations representing the actual tuple value.

Each tuple consists of a constant number of components.

Template Parameters
Domainthe domain of the component values
aritythe number of components within an instance
