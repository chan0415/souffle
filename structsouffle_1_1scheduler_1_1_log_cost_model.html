Detailed Description

A model computing the cost of a schedule based on the formula:

C = prod_{ a in schedule } ( n_a^(f_a/m_a) )

where C ... the costs, a ... atoms in a schedule, n_a ... cardinality of relation referenced by atom a f_a ... free arguments of atom a m_a ... arity of relation referenced by atom a
