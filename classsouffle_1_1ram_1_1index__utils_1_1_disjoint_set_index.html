Detailed Description

template<typename Index>
class souffle::ram::index_utils::DisjointSetIndex< Index >

A disjoint index is like a direct index storing the indexed elements directly within the maintained index structure.

However, unlike the B-Tree utilized by a direct index, a disjoint set is utilized, with pairs being implicit.

Template Parameters
Index.. the index to be internally utilized.
